Proceso de Compilacion y Ejecucion del Programa:

Requisitos:
-Consola Unix
 

1. Instalar/Actualizar tu version de makefile

Consola: `sudo apt-get install make`

2. Ubicarse en el directorio descargado

3. Generar el ejecutable "imdb"

Consola: `make imdb`

4. Ejecutar el archivo

Consola: `./imdb archivo.csv`
